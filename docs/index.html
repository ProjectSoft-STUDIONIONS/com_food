Компонент «ПИТАНИЕ» для Joomla! CMS

Компонент com_food для Joomla! CMS. Обслуживание директории food сайта школы. Актуально для сайтов школ РФ.

Описание

Компонент com_food для Joomla! CMS разработан для обслуживание директории food на сайтах школ Российской Федерации.

Установку компонента нужно провести через административную панель Joomla! CMS.

Скачать последнюю версию компонента для своей версии Joomla! CMS.

Для Joomla! CMS v3.x.x перейти на страницу установки расширений Меню -> Расширения -> Расширения -> Установка. Перейти на вкладку Загрузить и установить. Выбрать или перетащить в поле для загрузки только что скаченный файл для вашей системы. Дождаться конца установки компонента.

Для Joomla! CMS v4.x.x-v6.x.x перейти на страницу Система и в блоке Установка выбрать пункт Расширения. Перейти на вкладку Загрузить и установить. Выбрать или перетащить в поле для загрузки только что скаченный файл для вашей системы. Дождаться конца установки компонента.

Компонент установлен. Вам осталось только запустить компонент. Для этого нужно перейти в Компоненты и выбрать пункт ПИТАНИЕ.

Директория food присутствует всегда. Для дополнительных директорий, например индивидуального питания, нужно перейти в Настройки компонета. Кнопка настройки компонента присутствует на странице компонента ПИТАНИЕ вправом верхнем углу.

На вкладке Опции вы можете перечислить директории, которые нужно создать. Так же вы можете настроить автоудаление старых файлов, при этом вы можете выбрать временной интервал от одного года до пяти лет. Т. е. если выбран один год, то вы сохраните файлы за текущей год и за прошлый включительно.

Данная опция позволяет вам сэкономить объем используемого пространства выделенного системой вашего хост провайдера.

Так же вы можете настроить Права доступа для ваших Менеджеров. Помните, что не нужно давать доступ к компоненту в публичной части.

В систему так же добавляется ссылка на проверку обновлений для компонента com_food. При выходе новой версии вам будет предложено обновление компонента в соответствующем месте определённое вашей системой CMS.

Удачного использования компонента!
